# Pasta de Imagens

Esta pasta deve conter as seguintes imagens:

## Logo da Empresa

- **Arquivo**: `logo.png`
- **Formato**: PNG com fundo transparente
- **Dimensões recomendadas**: 200x200px (ou proporcional)
- **Uso**: Exibido no header e footer do site

## Favicon

- **Arquivo**: `favicon.png` ou `favicon.ico`
- **Formato**: PNG ou ICO
- **Dimensões**: 32x32px ou 16x16px
- **Uso**: Ícone exibido na aba do navegador

## Instruções

1. Adicione o logo da empresa AMA Recicláveis nesta pasta
2. Certifique-se de que o arquivo se chama exatamente `logo.png`
3. O logo deve ter fundo transparente para melhor integração visual
4. Para o favicon, você pode usar um ícone de reciclagem ou a inicial "A" da marca

## Ferramentas Úteis

- Para criar/editar logos: Adobe Illustrator, Figma, Canva
- Para otimizar imagens: TinyPNG, ImageOptim
- Para criar favicon: Favicon.io, RealFaviconGenerator
